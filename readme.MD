# Introduction
This is a code repository for our first frontend website. It serves as web-based hotel booking platform that displays some information relating to the hotel, including interior & exterior design, floor plan, facilities, etc. This was meant to complete our assignment in subject "Introduction to Web Programming". 

# Key Features
- Sticky Navigation: Navbar that stays on top while scrolling.
- Modern Layout: Using Flexbox & Grid .
- Responsive Interface: Mobile-friendly website interface.

# Tech Stack
- Frontend: HTML + CSS (Without framework)
- Version Control: Git
- IDE: VSCode
