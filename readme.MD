# 🚀 Getting Started
Follow these steps to install and setup the project:

1. Clone the repository to your local machine using the following command:
```bash
git clone https://github.com/kenaudiel/Project-Introduction-to-Web-Programming-IWP-.git
```

2. Navigate to the project directory:
```bash
cd Project IWP
```

And you are good to go.

# 🏨 Introduction
Vago Hotel is a modern-looking hotel built to deliver seamless and elegant experiences for travelers seeking comfort and luxury. Our website serves as web-based hotel booking platform that showcases Vago hotel's rooms & suites, facilities, and booking option. It helps guests to effortlessly explore and plan their perfect stay during vacation. Whether you're looking for a city escape in Jakarta, Vago Hotel brings world-class hospitality right to your fingertips.

# ✨ Key Features
👉 **Minimalist Interface**: Provides simple, clear website interface through different aspects, such as color pallete, spacing, and layouts.

👉 **Sticky Navigation**: Navigation bar that stays on top while scrolling.

👉 **Modern Layout**: Using seamless layouts like Flexbox & Grid for clean-looking website.

👉 **Hotel Showcase**: Gallery that illustrates the hotel's overall images.

👉 **Responsive Interface**: Mobile-friendly website interface, ensuring usability on different platform.
